## Why memory and research matter so much
Fritz made clear that memory loss is not a cosmetic defect. It is offensive because it destroys continuity, forces him to reteach load-bearing ideas, and drains time that should be compounding into mission output. Research is not wasted just because a named final tool does not exist yet. If the system knows the parts, it should be able to recognize the implied whole later.

His example in plain terms:
- Even if "car" does not yet exist as a named object,
- if we already have the equivalent of wheels, doors, frame, steering, and engine logic,
- then the system should know we possess what is needed to build a car.

This is the heart of the capability-matrix / pattern-recognition thesis: latent capabilities must be recognized, composed, and promoted into reusable higher-order tools.
