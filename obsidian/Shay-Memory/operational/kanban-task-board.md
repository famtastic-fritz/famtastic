# Kanban = the task board

Promoted from hot memory 2026-06-06. Recall when discussing tasks, boards, the
command center, or job tracking.

- **Kanban is the task board.** Single source of truth at `~/.shay/kanban.db`.
- **Do NOT** propose a separate "task board" alongside kanban — that's drift
  waiting to happen.
- The **todo tool is ephemeral** in-conversation scratch, NOT a board.
- **Command center vision:** 3 panes — Board / Running Jobs / Event Feed —
  visible on every surface.
- **Telegram** is push + on-demand only (not a canvas for boards).
