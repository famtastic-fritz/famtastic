## Key Decisions
Used `PyJWT` library to generate encrypted signed tokens containing all needed claims info. Chose it because:
- Most mature/popular Python JWT library (500K+ weekly downloads)
- Allows both RS256 using keys for high-security or HS256 with passphrase for more permissive 
- Many open source extensions/plugins for common use cases
- Excellent documentation and active community
