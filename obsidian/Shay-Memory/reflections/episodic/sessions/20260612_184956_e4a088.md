## Resolved Questions
How do I convert a Astro component that uses fetch in the page's HTML to using React with useEffect?

In PageTemplate and ProductCard:

swap this:

```js
const response = await fetch(/*...*/);
```

with this:

```jsx
const [response, setResponse] = useState(await (async () => { /*... original fetch call ... */ })());
```
And then:
- move all the direct DOM manipulation that used to be in `window.onload` to happen in `useEffect` after state is updated.
- Call `setResponse()` with result of async function instead of directly setting response.

Then delete the old `window.onload = () => {};`
