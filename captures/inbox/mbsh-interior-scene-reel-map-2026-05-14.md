# MBSH Interior Scene Reel Map — 2026-05-14

Purpose: treat every interior page as one orchestrated interactive movie reel, not independent web pages. RSVP is the prototype grammar; the same hero + mini-marquee + note/info panel + bridge/bleed + form/control language rolls across the remaining scenes.

## Global grammar

- First viewport is one opening composition: layered full-width hero plus lower-third mini-marquee scene marker.
- Scene marker carries only scene number and cinematic title. No date/time/location line.
- Tier 1 Harry appears in the hero as the live-action photoreal mascot treatment. This is the premiere-world Harry.
- Tier 2 Harry remains for advisor/chatbot moments later.
- Tier 3 Harry remains for normal UI, forms, section callouts, and small scene decoration.
- Every hero bleeds into the next scene with page-specific physical/lighting artifacts.
- Desktop can carry more layers and motion. Mobile collapses to one strong hero, one simple bridge object/light trail, readable Harry, and lighter motion.
