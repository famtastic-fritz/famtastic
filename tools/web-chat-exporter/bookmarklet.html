Claude Chat Exporter

One-click export of any claude.ai conversation to markdown for the FAMtastic capture flywheel.

Export Claude Chat
↑ Drag this button to your bookmarks bar

Use it

  1. Drag the button above to your bookmarks bar
  2. Open any claude.ai conversation
  3. Click the bookmark → a new tab opens with the full chat as markdown
  4. Cmd+A, Cmd+C the markdown
  5. Save to ~/famtastic/imports/web-chats/YYYY-MM-DD_<slug>.md
  6. Run the capture tool: node lib/famtastic/capture/cli.js scan

If it breaks

Claude Web's DOM updates over time. The bookmarklet tries three selector patterns. If none match, it shows an alert and you fall back to Option C in the README — manual select-all → copy → save → normalize.

Why a bookmarklet

Zero install. Works in any browser. No extension permission to approve. The tradeoff: limited capability vs. a real extension. For MVP that's the right tradeoff.
