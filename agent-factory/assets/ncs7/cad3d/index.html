NCS 3D CAD Viewer — Floor Plan Exploded into Layers

An architectural floor plan lifted off the sheet and exploded into stacked floor plates in space. Level 2 splits further into its discipline layers — Architectural, Structural, MEP — the same separation the AIA CAD Layer Guidelines define. Drag to orbit, scroll to zoom.

How a real CAD PDF feeds in

  1. Load the PDF with pdf.js: pdfjsLib.getDocument(url).
  2. Render each page to an offscreen <canvas> via page.render({ canvasContext, viewport }).
  3. Wrap the canvas as a THREE.CanvasTexture on a PlaneGeometry.
  4. Stack the planes by sheet / level (A-101, A-102…), offset on Y, then orbit — exactly the layout this demo builds with line geometry.

True layer extraction: CAD-authored PDFs carry Optional Content Groups (OCGs) — the PDF equivalent of CAD layers. page.getOptionalContentConfig() lets you toggle each OCG and re-render per layer, producing one texture per discipline (A-WALL, S-COLS, M-HVAC…) → the Architectural / Structural / MEP explosion shown here. For vector accuracy, walk page.getOperatorList() and re-emit drawing ops per layer. Either path: canvas → texture → plane → stack → orbit.
