# FAMtastic Designs — Campaign Plan

> The concrete first campaign: lanes, cadence, budget, and stop/go gates.
>
> Generated by the Agent Factory · 2026-06-18 · internal use only


## Primary lane (run this first, exclusively)
**"Conversion Leak / Checkout Rescue" diagnostic campaign.** One offer, one
message, two outreach channels, one paid entry point. Do not split focus until
this lane produces cash.

## Backup lane (activate only if primary stays dead 48h)
**"Google Ads / Tracking Leak Audit" — $149.** Same machine, different pain.
Kept on the shelf, not run in parallel — parallel lanes dilute the test.

## Campaign structure
- **Audience A — Agencies:** offer = white-label overflow builds + rescue.
- **Audience B — Merchants:** offer = $500 checkout/funnel rescue diagnostic.
- **Audience C — Local services:** offer = $149 audit → Launch build.

## Cadence (per business day, first 2 weeks)
- 10 tailored Upwork/marketplace proposals before noon.
- 10 direct emails + 10 LinkedIn DMs from the 25-target daily list.
- 100% of replies answered within 2 working hours.
- 1 proof artifact published every 2 days.

## Budget plan
- **Weeks 1–2: $0 paid.** Pure outreach. Cost is time, not cash.
- **Gate 1 (after first paid job):** turn on $10–20/day to a single audit
  landing page. Kill any ad set under a 2% landing-conversion after $100 spend.
- **Gate 2 (after 3 paid jobs):** add retargeting only once real traffic exists.
- **Hard rule:** ad spend never exceeds 20% of trailing-30-day collected revenue.

## Stop / go gates
| Gate | Condition | Action |
|------|-----------|--------|
| G0 | 0 replies in 48h on primary lane | activate backup lane in parallel |
| G1 | 1 paid diagnostic closed | switch on small paid test, build case study |
| G2 | 3 paid jobs collected | add retargeting + raise build prices 10% |
| G3 | CAC > 25% of order value for 7 days | pause paid, return to pure outreach |

## First-10 execution actions (today)
1. Lock the offer copy (one sentence) — done in the sales deliverable.
2. Publish the one-page sales doc (scope, price, CTA, turnaround).
3. Finalize the 6-point diagnostic checklist.
4. Two proposal templates ready: merchant-direct + agency-overflow.
5. Stand up the reusable Upwork/marketplace proposal block.
6. Shortlist 20 relevant jobs (Authorize.Net, checkout fix, WooCommerce
   payment, payment-gateway cleanup, "site redesign").
7. Send 10 tailored proposals before noon.
8. Build the 25-target outreach list (agencies + merchants).
9. Send 10 direct emails + 10 DMs offering the diagnostic.
10. Prep the Loom template so delivery starts the same day someone bites.
