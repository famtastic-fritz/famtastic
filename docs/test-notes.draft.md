# Draft Notes from ChatGPT

## Overview
This is a test document ingested from ChatGPT into the Hub using the two-way sync functionality.

## Key Points
- Two-way sync enables ChatGPT → Hub ingestion
- Files are written to safe directories (docs/, convos/)
- All ingests are logged with UUID and timestamp
- Default behavior creates .draft.md files for safety
