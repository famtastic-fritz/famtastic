# The economics that change my routing

Cookbook serves VRAM-fit local models at zero marginal cost. That flips the default for a whole class of swarm work.

Route to my local Odysseus

  • Bulk / fan-out worker tasks
  • Low-stakes drafts & transforms
  • Retry storms & self-heal loops
  • Anything privacy-sensitive
  • Offline / API-down resilience

Keep on cloud (Opus)

  • High-reasoning council judges
  • Large-context synthesis
  • Final-quality client output
  • Adversarial verification gates
