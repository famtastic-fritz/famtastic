Rail icon hover-state prototype

Default state is dim. Hover lights the icon warm, scales it 1.06×, drops a soft glow, and slides a glass label out from the rail. The active item (Sites here) carries a 2px warm inset bar on the left and stays full opacity. The disabled item (Media) sits at 32% opacity and ignores hover.

Defaultcolor: text-2 / opacity 0.85 / no scale
Hovercolor: glow-warm / scale 1.06 / drop-shadow / label slides
Activecolor: glow-warm / inset bar 2px / no scale
Disabledcolor: text-3 / opacity 0.4 / no hover

Icons here are placeholders to demonstrate the interaction. Final glyphs come from each domain's brief (sites.md, brainstorm.md, plans.md, components.md, media.md, research.md, admin.md).
