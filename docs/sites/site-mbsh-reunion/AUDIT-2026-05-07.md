## 6. Through-the-Years recommendation

### What was wrong
The five era cards used `assets/story/era-1926-1959.jpg` ... `era-2026.jpg`,
which on inspection are **abstract AI-generated decorative imagery** — red
and silver ribbons, cyan light tunnels, glossy magazine textures. They tell
no story about MBSH, no era, no class, no Miami Beach. The captions were
thin one-liners. There is no horizontal filmstrip pin animation that the
original Design Map called for; just a vertical stack of decorative panels.

### Options weighed
1. **Rebuild now** — needs real archival photos / yearbook scans / programs;
   not in hand tonight.
2. **Disable for production** — kills the memory submission funnel.
3. **Simplified for launch** ← **chosen**
4. **Coming-soon page** — same intent as 3, slightly more aggressive.

### What landed
A poster-style "A century of Hi-Tides" panel: Cinzel eyebrow, "1926 → 2026."
display headline, italic copy explaining the reel is in production, a dotted
roman-numeral list of the five eras (`I. 1926–1959 · The first Hi-Tides`,
through `V. 2026 · One hundred years. Still rising.`), and a Cutive-mono
"Coming soon — the committee is curating photos and programs" note. The
existing memory submission form below remains active, so the funnel stays
open. Harry `22-walk-frame` still anchors the page via JS injection.

The Where-Next on every other page still routes through Through-Years
because the page is honest about what it is (in production), not broken.

### Production-safety check
The page now: matches the universe vocabulary, sets clear expectations,
keeps the form active, and doesn't ship broken or generic visuals.
