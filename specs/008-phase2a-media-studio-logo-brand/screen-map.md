# Phase 2A Screen Map — Media Studio Logo Lab

## 1. Brand Projects
Shows active brand projects, latest approved asset, open tasks, cost used, next action.

## 2. Logo Lab Intake
Fields: brand name, origin story, old logo uploads, liked concepts, disliked concepts, audience, tone, production needs.

## 3. Concept Board
Pinterest/Figma-like board of references, old marks, AI WIPs, notes, tags, and visual themes.

## 4. Prompt Lab
Prompt editor with variables, negative prompts, provider/model target, estimated cost, seed/version history, research snippets.

## 5. Generation Plan / Provider Queue
Dry-run first. Shows MuAPI/Open Generative/manual/Adobe/Remotion options, cost, expected output, approval gate.

## 6. Variant Grid
Cards for each asset: image, source, prompt, provider, status, rating, notes, lineage, compare checkbox.

## 7. Critique & Compare
Side-by-side selected variants with rubric: legibility, distinctiveness, scalability, FAMtastic DNA, dark/light fit, production readiness.

## 8. Refinement / Vector Pass
Tracks vectorization/redraw tasks, cleanup steps, SVG/PDF/export readiness, manual handoff notes.

## 9. Usage Test Board
Drops logo into nav, favicon, social avatar, dark hero, light doc, merch/signage mockup, animation bumper.

## 10. Brand Kit Export
Exports DESIGN.md, tokens, SVG/PNG/WebP/PDF, lockups, clear-space rules, color/type/motion rules, prompt history.

## 11. Prompt/Asset Lineage
Timeline of every prompt, model, output, critique, approval, file, and where-used link.

## 12. Post-Eval
Captures what became reusable: skill, recipe, prompt pattern, component need, tool gap, process change.
