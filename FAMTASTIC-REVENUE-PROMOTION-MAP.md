## 2. Cross-Sell Map

```
Hosting customer
  └── sees Designs cross-promo on homepage / checkout
      └── buys website → gets hosting bundle upsell
          └── sees Thoughts articles about SEO/growth
              └── upgrades hosting / buys automation

Designs lead
  └── receives proof preview
      └── claim page offers hosting bundle
          └── onboarding links to Thoughts lessons
              └── retainer / automation upsell

Thoughts reader
  ├── article about hosting → Hosting affiliate/checkout
  ├── article about design → Designs preview form
  └── article about tools → Sponsored / affiliate revenue
```

### Concrete cross-sell placements

| Source page | Placement | Destination | Goal |
|---|---|---|---|
| Hosting homepage | "Need a site to go with your hosting?" banner | `/` or `/services` on Designs | Design lead |
| Hosting checkout | "Add a logo + one-page site" upsell | Designs checkout | Increase AOV |
| Hosting dashboard | "Grow your site" article strip | Thoughts collection | Engagement + affiliate |
| Designs homepage | "Built sites need a home" banner | Hosting plans | Hosting sale |
| Designs campaign page | "Add hosting + domain" bundle toggle | Hosting checkout | Bundle sale |
| Designs proof gallery | "How we host these sites" link | Thoughts article | Authority + affiliate |
| Thoughts article | Contextual product CTA | Hosting / Designs | Convert reader |
| Thoughts homepage | Featured collections + product cards | Both properties | Discovery |

## 3. Promotion Plan (Phased)

### Phase 0 — Foundations (now)

1. **Hosting homepage editable + cross-promo live** ✅
2. **Hosting backend wired** ✅ (live PayPal, GoDaddy API, cart, auth)
3. **Designs content model + sample collection flow** ✅
4. **Thoughts content model + capture flow** ✅
5. **GoDaddy real provisioning** — next hosting hardening task
6. **Set up Google Search Console + GA4** on all three domains
7. **Verify cross-links work** and pass UTM params (`?ref=designs`, `?ref=thoughts`, `?ref=hosting`)

### Phase 1 — Hosting traction (2-4 weeks)

1. **SEO:** Publish 5 hosting-related Thoughts articles targeting low-competition keywords (`custom nameservers`, `white label hosting`, `small business web hosting Miami`).
2. **Local SEO:** Create location pages for Miami, Atlanta, and 3 target cities; list in local directories.
3. **Email capture:** Add a "site speed check" lead magnet on Hosting homepage; feed into Designs pipeline.
4. **Referral program:** Hosting customer gets $25 credit for referring a design client.
5. **Paid retargeting:** Install Meta pixel + Google Ads tag; retarget homepage visitors with Small Biz Starter bundle.

### Phase 2 — Designs pipeline (4-8 weeks)

1. **Proof Factory live:** Generate preview pages from URL + category.
2. **Local Services campaign:** Target 100 businesses; send proof previews; book calls / close pay-now packages.
3. **Church / nonprofit campaigns:** Run calibrated outreach with high-sensitivity review queue.
4. **Case studies:** Publish 5 before/after proofs as Thoughts articles with Designs CTAs.
5. **Partnerships:** Cross-promote with complementary service providers (bookkeepers, photographers, marketers).

### Phase 3 — Automation + app revenue (8-16 weeks)

1. **Lead Scout extension:** Ship Chrome extension MVP; charge $9/mo for unlimited scans.
2. **Thoughts Capture app:** Ship voice-to-publish iOS/Android app; freemium.
3. **Hosting Status Widget:** Android widget MVP; upsell white-label version to resellers.
4. **Auto-migration assistant:** One-click WordPress migration; charge per use or include in higher tiers.
5. **Affiliate program:** Recruit bloggers / agencies to promote Hosting/Designs for recurring commissions.

### Phase 4 — Scale (ongoing)

1. **Autonomous revenue pipeline:** Connect Scout → Build → Close agents for service arbitrage.
2. **Paid ads for high-LTV products:** Google Ads for hosting keywords; Meta for design services.
3. **Email newsletter:** Weekly FAMtastic Digest with articles + deals.
4. **Community / membership:** Paid community for agency owners using white-label hosting.
5. **Marketplace:** List design templates / automation workflows for sale.
