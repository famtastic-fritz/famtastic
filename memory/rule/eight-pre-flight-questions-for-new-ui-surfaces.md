# Eight pre-flight questions every new UI surface answers before drafting

The standard checklist: (1) Is this a creation tool or a website? (2) What page type is this? (3) Where does Shay live? (4) Does this surface push the canvas or float over it? (5) Per-domain coherent icon set or generic library mixed? (6) What 2 product references prove this layout works? (7) What does this look like in empty/loading/single-result/many-results/error? (8) What would Codex adversarial review say? Add to docs/operating-rules/.
