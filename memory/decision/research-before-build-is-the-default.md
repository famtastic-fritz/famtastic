# Research-before-build is the new default for every page-type-shaped problem

The previous Workbench trajectory built layout before researching the page type. The reversal (pause implementation, do per-page research, return with mockups) produced six designed page types in one session. Research-before-build is now the default. Build-first only when the surface clearly fits an already-researched page type.
