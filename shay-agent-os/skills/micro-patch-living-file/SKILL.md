# Micro Patch Living File

Use when editing a file that is continuously appended/regenerated by a process (DNA files, generated bindings, ledgers). Read first, make the smallest exact-match edit, never full-rewrite a file another process owns, and verify the closing/structural tokens survived your edit.

## When to use
Surgically patch a long-lived auto-updated file without clobbering it
